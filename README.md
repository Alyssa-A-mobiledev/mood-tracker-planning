# Daily Mood Tracker

## Project Description
The Daily Mood Tracker is a simple web application that allows users to log their daily mood and optional notes.
The goal of the app is to help users reflect on emotional patterns over time by saving and displaying their mood history.

## Planned Features
1. Log daily mood (happy, sad, stressed, etc.)
2. Add optional notes for each mood entry
3. Store mood entries using localStorage
4. View mood history on the page
5. Delete or edit mood entries

## Tech Stack
- HTML — Page structure and content
- CSS — Styling and layout
- JavaScript — App logic and localStorage interaction
- localStorage - To store mood entry

## Deployment
This project runs entirely in the browser and will be deployed using Netlify.
